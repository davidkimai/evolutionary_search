- schema validation
- scoring math
- dedupe behavior
- replay run completes

## Stronger tests
- mixed known/unknown evidence handling
- ranking differs across profiles
- replay/live contract parity
- timeout / provider failure handling

- one command starts replay demo
- one command exports markdown dossier
- run page loads without live credentials
- top result contains visible evidence
