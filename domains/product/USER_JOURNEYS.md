# User Journeys

## 1. Hero search flow
User inputs:
- who they are
- what they are seeking
- sectors / geography
- constraints

System:
- launches a run
- fans out across sources
- streams progress
- returns a ranked shortlist

Success:
- top opportunities are useful immediately

## 2. Opportunity detail flow
User opens one opportunity and sees:
- title
- source
- deadline
- reward / amount
- geography
- eligibility
- evidence snippets
- why it fits
- confidence
- next step

Success:
- user trusts the recommendation

## 3. Replay demo flow
Judge sees:
- no credentials setup pain
- run starts instantly from replay
- progress appears live
- shortlist lands cleanly

Success:
- demo is stable under pressure

## 4. Export flow
User exports:
- markdown dossier
- JSON
- shortlist summary

Success:
- output is ready to send or act on
